AOI - Abstract Object Inference Tool

Usage: aoi [options] <input dir> <output dir>

AOI analyzes EO programs to infer object types. It finds .xmir files
in the input directory and generates .xml files with type information.

Arguments:
    <input dir>     Directory with .xmir files (searches recursively)
    <output dir>    Output directory for .xml files with inferred types

Options:
    --help          Print this message and exit.
    --version       Print the version of aoi and exit..
